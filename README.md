# 🎓 CampusConnect

CampusConnect est une plateforme universitaire moderne visant à digitaliser et centraliser les services académiques.

## 🚀 Objectif

Améliorer l'expérience universitaire en offrant :
- Une communication fluide
- Un accès rapide aux informations
- Une gestion centralisée des services

---

## 📱 Plateformes

CampusConnect est disponible sur deux plateformes distinctes :

### 🌐 Version Web
- Framework : Next.js
- Repository : https://github.com/ton-repo/campusconnect-web

### 📲 Application Mobile
- Framework : Flutter
- Repository : 
https://github.com/thioyehassimiou-source/campus-connect.git
---

## 🛠️ Technologies

### Web
- Next.js
- React
- Tailwind CSS

### Mobile
- Flutter

### Backend
- NodeJS express (Auth + Base de données)

---

## 🔑 Fonctionnalités principales

- Authentification (Email / Google)
- Tableau de bord étudiant
- Fil d’actualités
- Notifications
- Gestion académique
