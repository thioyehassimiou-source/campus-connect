# 🎓 CampusConnect

CampusConnect est une plateforme universitaire moderne visant à digitaliser et centraliser les services académiques.

## 🚀 Objectif

Améliorer l'expérience universitaire en offrant :
- Une communication fluide
- Un accès rapide aux informations
- Une gestion centralisée des services

---

## 📱 Plateformes

CampusConnect est disponible sur deux plateformes distinctes :

### 🌐 Version Web
- Framework : Next.js (App Router, Server Actions, SSR)
Base de données : PostgreSQL via nodejs
Sécurité : Protection au niveau de l'authentification et sécurité au niveau des lignes (RLS)
UI/UX : Design MD3 (Material Design 3) avec animations fluides.
- Repository : 
https://github.com/thioyehassimiou-source/campusconnect-web.git
### 📲 Application Mobile
- Framework : Flutter
- Repository : 
https://github.com/thioyehassimiou-source/campus-connect.git
---

## 🛠️ Technologies

### Web
- Next.js
- React
- Tailwind CSS

### Mobile
- Flutter

### Backend
